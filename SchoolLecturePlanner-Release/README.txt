School Lecture Planner

Gereksinimler:
- Java 17 veya üzeri (https://adoptium.net/temurin/releases/?version=17)
- Windows işletim sistemi

Kurulum:
1. Java 17'yi yükleyin
2. Klasörü istediğiniz bir konuma çıkartın
3. SchoolLecturePlanner.exe'yi çalıştırın

Not: 
- exe ve lib klasörü aynı dizinde olmalıdır
- Veri dosyaları data klasöründe bulunmaktadır
